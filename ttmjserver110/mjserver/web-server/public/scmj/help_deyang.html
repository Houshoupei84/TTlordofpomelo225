德阳麻将玩法说明

一、游戏规则

1.游戏人数：4人

2.游戏用筒、条、万三种牌，共108张。

4.必须缺门才可胡牌，即胡牌时不能有三种花色的牌。

5.玩家胡牌后，其余未胡牌玩家继续游戏，直到流局或只有一位未胡玩家为止。

6.只剩四张牌时必须胡牌，不能过胡。

7. 坐庄规则：上一局最先胡牌的为庄，如一炮多响则点炮的玩家为庄，如无任何人胡牌则上一局的庄家的下家为庄

二、术语解释

1.定缺

  摸牌结束后由玩家选择一种花色作为该局游戏定缺的花色。游戏中必须打完该花色才能胡牌。（即手上不能留定缺花色的牌）

2.刮风下雨

  1）刮风（明杠）：分直杠和面下杠（弯杠）

直杠：玩家手中有一个暗刻（手中又三张一样的牌），当其他玩家打出该暗刻牌时，该玩家选择杠牌。直杠时立刻收取引杠者两倍底金。

面下杠：也叫弯杠，即玩家手中有一个明刻（碰出的牌），自己又摸到该明刻的牌，选择杠牌。面下杠时立刻收取其余未胡玩家一倍底金。其余玩家若抢杠胡，则面下杠玩家要退回其余未胡牌玩家一倍底金。

 2）下雨（暗杠）：下雨：指玩家手中有四张牌（未碰），此时玩家选择杠牌。下雨立刻收取其余未胡玩家2倍底金，不可被抢杠胡。

3.查大叫

流局时没叫的玩家赔给有叫的玩家最大可能番数（大叫），并退回全部刮风下雨所得。

4. 过手胡：如果某一回合种，有人点炮，该玩家在第一次没有胡，则其他玩家在该回合中再次点炮时，只要没有加番该玩家就不能胡（自摸除外）。当该玩家有动牌（即碰、杠、摸牌）后，该回合过手胡限制解除。

5.自摸加底

玩家自摸胡牌时，除获得相应牌型的积分外，还要收取未胡玩家一倍的底金积分。

6.自摸加番

玩家自摸胡牌时，需要在胡牌的牌型上加一番。

7.点杠花（点炮）、点杠花（自摸）

    即玩家点杠后杠上花胡牌时，如果选择点杠杠上花（点炮），按自摸算但只收点杠者一家分数；如果选择点杠杠上花（自摸），按自摸算并收取三家分数。

弯杠杠上花和暗杠杠上花按照自摸计算分数。

三、胡牌类型

1.平胡（0番）：四坎牌加一对将

2.对对胡（1番）：四副刻子加一对将

3.清一色（2番）：全部是一种花色的平胡

4.带幺九（2番）：每副顺子、刻子、将牌都包含一或九

5.七对（2番）：玩家手牌都是两张一对的，没有碰和刮风下雨

6.龙七对（3番）：牌型为七对，但玩家手中有四张一样的牌（没有杠出）

7.清对（3番）：清一色＋对对胡

8.将对（3番）：由2、5、8组成的对对胡

9.清七对（4番）：清一色＋七对

10.清龙七对（5番）：清一色＋龙七对

11.天胡（5番）：庄家起牌后胡牌

12.地胡（5番）：在第一回合中，非庄家自摸胡牌

四、分数计算

1.基本输（赢）＝底金＊倍数＋刮风下雨。

2点炮收取点炮玩家的积分（一炮多响一对一结算）；自摸收取所有未胡玩家的积分。

3.另加番

［根］：＋1番，有四张一样的牌（无论是否碰、杠）可重复计算。

［杠上花］：＋1番，玩家杠牌后，补的牌恰好胡牌。

［杠上炮］：＋1番，玩家杠牌后，补牌后打出一张牌，打出的牌被其他玩家胡牌。此时退回刮风下雨所得。

［抢杠胡］：＋1番，玩家（刮风）面下杠时，其他玩家可以抢杠胡，刮风下雨的积分要退还给各个玩家。

［扫底胡］：＋1番，牌墙最后一张牌自摸或点炮胡。

［金钩胡］：＋1番，在［对对胡］一类的牌型中，下叫的牌为将牌，且其余牌已经碰、杠出，此时胡牌总番＋1番。

［海底胡］：＋1番，牌墙最后一张牌点炮胡。

［断幺九］：＋1番，胡牌的时候没有1和9。

［夹心五］：＋1番，胡的牌是五（条筒万均可），并且是由四和六夹胡五。

［姊妹对］：＋1番，胡牌牌型中有相同花色三个连续成对的序数牌

［一条龙］：＋1番，胡牌牌型中必须包含同种花色的1至9组成的三个顺字、一坎牌和一对将。

备注：各个牌型不重复计算，只收取最大番。
