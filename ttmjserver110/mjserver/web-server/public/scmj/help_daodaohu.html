倒倒胡玩法说明

一、游戏规则

1.游戏人数：4人

2.游戏用筒、条、万三种牌，共108张。

3. 坐庄规则：游戏中采用连庄方式，仅第一局为创建房间的玩家为庄，之后将由本局胡牌的玩家下局坐庄，如庄家胡牌，则继续连庄。若荒庄则摸最后一张牌的玩家下局坐庄。

4.无大胡，所有牌型都按小胡分数计算。

二、术语解释

1.自摸：所胡牌为自己正常摸得，即称之为自摸胡，简称“自摸”。

2.明杠：明杠分为两类，一类为自己手中已有三张相同牌，此时另外的玩家放出一张该牌，玩家即可选择杠牌，此明杠为对方放杠。另一类杠为玩家又摸到一张自己已碰过的牌，选择杠后为摸明杠。

3.暗杠：自己摸得四张（四张全部为玩家摸得）一样的牌后，选择杠后称之为暗杠。抢杠胡由杠的玩家付积分，按点炮算，暗杠不能抢。

4.抢杠：玩家明杠时，有人正胡其所杠这张牌，即称之为“抢杠胡”，简称“抢杠”。

5.荒庄：摸完所有牌后都无人胡牌即称为荒庄。

6. 一炮多响：如果同时有两家以上叫相同的牌, 若有人杠这张牌或杠后打出该牌, 该两家可以同时胡, 称 “一炮双响” 或 “一炮三响”。分数由点炮玩家一对一结算。

三、分数计算

1.明杠：每家收1分

2.暗杠：每家收2分

3.点炮：点炮者出1分

4.自摸：每家收2分
