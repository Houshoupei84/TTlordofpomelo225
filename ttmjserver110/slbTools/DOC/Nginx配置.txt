以下是NGINX的 嘉兴乐玩配置
1. 请求限制
	limit_req_zone
	 limit_req_zone $client_ip zone=one:20m rate=50r/s;
	zone 限制空间大小, 每个基本上180b左右 , rate 限制次数 /s
	limit_req zone=one nodelay;  持续1秒就进行限制
	limit_req zone=one burst=5;	持续5秒进行限制
2. 连接限制
	limit_conn_zone $client_ip zone=addr:20m;	// 如上
    limit_conn_status 403;	// 如上
	
	
	limit_conn addr 50;    // 限制次数， 50次每秒
3. 访问限制
	if ($uri !~* .(json|html|zip|manifest|txt|project|png|dat)$) {
            return 403;
        }
	正则匹配拓展名, 如果不是,直接抛掉
4. 系统
	1. worker_connections 1024; 每个进程处理的数量
	2. worker_processes auto; 进程自动， 可以设置为别的
	3. user root;	使用 root权限
